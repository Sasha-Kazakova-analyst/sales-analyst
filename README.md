
# Анализ выручки по городам

## 📌 О проекте
Проект демонстрирует работу с большими данными:
- генерация тестового файла (50 000 записей)
- обработка по частям (chunking)
- расчёт общей выручки и выручки по городам
- визуализация результатов
- сохранение итогов в CSV и PNG

## 🛠 Технологии
- Python
- pandas
- matplotlib
- Google Colab

## 🚀 Как запустить
1. Открыть в Google Colab
2. Запустить все ячейки
3. Файлы сохранятся и скачаются автоматически

## 📊 Результат
- График выручки по городам
- CSV-файл с итогами

## 📁 Файлы
- sales_analysis.py — код проекта
- itog_vyruchka.parquet — итоговая таблица
- vyruchka_po_gorodam.png — график
